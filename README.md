# Q3 — Date/Time Notification (Notification Channel)

## CSE 225: Developing Android Apps — CA1

### Features
- DatePickerDialog + TimePickerDialog for schedule selection
- Notification Channel (IMPORTANCE_HIGH) created at startup
- High-priority notification with BigTextStyle sent to system tray
- Runtime POST_NOTIFICATIONS permission handled (Android 13+)

### How to Run
1. Open Android Studio → File → Open → this folder
2. Gradle sync → Run ▶
3. Pick date, pick time, tap "Send Notification"

### Package
`com.example.datetimenotification`
